# Chocolate-Sales-Analysis
Multi-year sales performance analysis using Power BI and DAX.

Objective
Analyze multi-year revenue performance to identify growth drivers and investigate performance volatility.

Key Analysis
-Year-over-Year revenue growth
-Quarter-over-Quarter comparison
-Country-level volatility assessment
-Product contribution breakdown
-Investigation of Q4 2024 revenue decline

Key Insights
-Across the three-year period, sales consistently increase from Q1 to Q2 before declining through Q4. This pattern is observed across all  regions, suggesting seasonality or internal operational factors or incomplete reporting rather than market-specific influences.
-Revenue distribution appears well balanced, with no significant concentration in a small number of countries or products, indicating a  diversified performance portfolio.
-Australia remains the leading revenue contributor throughout the period, while Smooth Silky Salty consistently ranks as the top-performing  product.

Tools Used Power BI | DAX | SQL | Excel
